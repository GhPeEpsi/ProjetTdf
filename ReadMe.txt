Bonjour,
Bienvenue dans le code source de notre projet PHP sur le tour de france.
Les fichiers sont rangés dans des dossiers par types de langage (css/html/js/php/sql/texte(documentation)).
Les dossiers css et js, comportant les fichiers sources importé pour le fonctionnement du framework bootstrap que nous avons utilisé pour ce projet.
Dans le dossier html on peut retrouver les fichiers html.
Dans le dossier php on peut retrouver les fichiers php.
Presque (certain ne sont que des fichiers back-end) chaque fichier php inclut un fichier html qui permet "l'interface graphique".
Dans le dossier sql on retrouve le fichier dans lequel sont stockées les requetes mises au point pour cette application.
Dans le dossier texte on retrouve une partie de la documentation correspondante à notre projet.
Enfin index.html permet au projet de se lancer automatiquement en entrant dans le dossier de notre projet.
